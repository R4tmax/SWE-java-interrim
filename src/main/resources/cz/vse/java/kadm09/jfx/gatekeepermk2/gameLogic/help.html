GATEKEEPERMK2

What is the game about?

Game loosely interprets the story from my personal novel currently titled GATEKEEPER, that I am writing in my spare time.

How to play the game?

Player character

The game is played from the perspective of an anonymous knight. Who besides being clad in plate armour and wielding a greatsword. Is also a capable mage. Furthermore, the player character can withstand usage of potions and herbs and modify his gear in the field

Basic input

This variant of the game accepts input through the text field at the bottom of the panel. Game returns output into the text area in the right top. At any given moment, available commands can be accessed by typing HELP into the text field. I suggest hitting HELP whenerver you are not sure how to continue. Also, take time to learn the commands At the start of the game!. Incorrect inputs are ignored and warning given to the player.

Player information

There are two main areas to pay attention to. Player stats are always visible on the left screen, gamestate is always displayed on the icons in the middle of the screen. These icons correlate to the expected inputs, pay attention to them! If your health drops below 0, you die!

GUI specific interactions

Middle of the panel is dedicated to the quick input system of the game.

Quickmove section

On the right, there is a arrow section dedicated to moving quickly around the map. These commands correlate to using the MOVE -> DIRECTION command sequence in the CLI. Middle button displays information available about the room. Take note that player cannot be moved during combat.

Quickcast section

Below the gamestate Icons, quickcast menu is available. Pressing the button immediately attempts to cast the spell associated

Inventory bar

Finally, on the left, there is a visual representation of items held. Clicking the list item uses the item immediately.
